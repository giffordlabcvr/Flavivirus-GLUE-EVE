Welcome to the GitHub repository for **Flavivirus-GLUE-EVE** - a **[GLUE](http://glue-tools.cvr.gla.ac.uk)** project focussing on endogenous flaviviruses!

Flavivirus-GLUE-ERV extends **[Flavivirus-GLUE](https://github.com/giffordlabcvr/Flavivirus-GLUE)** through the incorporation of endogenous flaviviral elements (EFVs).

Please see the **[User Guide](https://github.com/giffordlabcvr/Flavivirus-GLUE/wiki)** for further details.
